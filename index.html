Atajos de Teclado en Microsoft Word

Estos atajos te ayudarán a trabajar más rápido con documentos de texto.

  • Ctrl + C: Copiar el texto o elemento seleccionado.
  • Ctrl + X: Cortar el texto o elemento seleccionado.
  • Ctrl + V: Pegar el texto o elemento copiado/cortado.
  • Ctrl + Z: Deshacer la última acción.
  • Ctrl + Y: Rehacer la última acción.
  • Ctrl + N: Crear un nuevo documento en blanco.
  • Ctrl + A: Seleccionar todo el contenido del documento.
  • Ctrl + G: Guardar el documento actual.
  • Ctrl + P: Abrir la ventana de impresión.
  • Ctrl + B: Poner el texto seleccionado en negrita.
  • Ctrl + K: Poner el texto seleccionado en cursiva.
  • Ctrl + S: Subrayar el texto seleccionado.
  • Ctrl + E: Centrar el párrafo seleccionado.
  • Ctrl + J: Justificar el párrafo seleccionado.
  • Ctrl + L: Alinear el párrafo seleccionado a la izquierda.
  • Ctrl + R: Alinear el párrafo seleccionado a la derecha.
  • Ctrl + F: Abrir el panel de búsqueda.
  • Ctrl + U: Abrir el panel de reemplazar.
  • Ctrl + T: Crear una sangría francesa.
  • Ctrl + D: Abrir el cuadro de diálogo Fuente.

Atajos de Teclado en Microsoft Excel

Estos atajos son útiles para navegar y editar hojas de cálculo.

  • Ctrl + C: Copiar la celda o rango seleccionado.
  • Ctrl + X: Cortar la celda o rango seleccionado.
  • Ctrl + V: Pegar la celda o rango copiado/cortado.
  • Ctrl + Z: Deshacer la última acción.
  • Ctrl + Y: Rehacer la última acción.
  • Ctrl + N: Crear un nuevo libro de trabajo.
  • Ctrl + A: Seleccionar toda la hoja de cálculo.
  • Ctrl + G: Guardar el libro de trabajo actual.
  • Ctrl + P: Abrir la ventana de impresión.
  • Ctrl + B: Poner el contenido de la celda en negrita.
  • Ctrl + K: Poner el contenido de la celda en cursiva.
  • Ctrl + S: Subrayar el contenido de la celda.
  • F2: Editar la celda activa.
  • F4: Fijar referencias de celda (alterna entre absolutas, relativas y mixtas) al editar una fórmula.
  • Ctrl + Espacio: Seleccionar la columna completa.
  • Shift + Espacio: Seleccionar la fila completa.
  • Ctrl + Inicio: Ir a la celda A1.
  • Ctrl + Fin: Ir a la última celda utilizada en la hoja.
  • Ctrl + Teclas de flecha: Moverse al borde de la región de datos actual.
  • Shift + Teclas de flecha: Extender la selección de celdas.
  • Ctrl + Shift + Teclas de flecha: Extender la selección hasta el borde de la región de datos actual.
  • Ctrl + ;: Insertar la fecha actual.
  • Ctrl + Shift + ;: Insertar la hora actual.
  • Ctrl + 1: Abrir el cuadro de diálogo Formato de celdas.

Atajos de Teclado en una Computadora HP (Windows)

Estos atajos son a nivel del sistema operativo y funcionan en la mayoría de las aplicaciones.

  • Ctrl + C: Copiar el elemento seleccionado.
  • Ctrl + X: Cortar el elemento seleccionado.
  • Ctrl + V: Pegar el elemento copiado/cortado.
  • Ctrl + Z: Deshacer la última acción.
  • Ctrl + Y: Rehacer la última acción.
  • Ctrl + A: Seleccionar todos los elementos en una ventana o documento.
  • Ctrl + F: Abrir la función de búsqueda en la mayoría de las aplicaciones.
  • Alt + Tab: Cambiar entre ventanas abiertas.
  • Alt + F4: Cerrar la ventana o aplicación activa.
  • Windows + D: Mostrar u ocultar el escritorio.
  • Windows + E: Abrir el Explorador de archivos.
  • Windows + L: Bloquear la computadora.
  • Windows + R: Abrir el cuadro de diálogo Ejecutar.
  • Windows + Tecla de flecha izquierda/derecha: Acoplar la ventana activa a la mitad izquierda/derecha de la pantalla.
  • Windows + Tecla de flecha arriba: Maximizar la ventana activa.
  • Windows + Tecla de flecha abajo: Minimizar la ventana activa o restaurar la ventana acoplada.
  • Impr Pant (o PrtScn): Tomar una captura de pantalla de toda la pantalla (se guarda en el portapapeles).
  • Windows + Impr Pant: Tomar una captura de pantalla de toda la pantalla y guardarla automáticamente en la carpeta "Capturas de pantalla" dentro de "Imágenes".
  • Ctrl + Shift + Esc: Abrir el Administrador de tareas.
  • Ctrl + Alt + Supr: Abrir la pantalla de opciones de seguridad de Windows (incluye Administrador de tareas, cambiar usuario, bloquear, etc.).
  • F5: Actualizar la ventana activa (común en navegadores web y Explorador de archivos).
