Atajos de Teclado en Microsoft Word

Estos atajos te ayudarán a trabajar más rápido con documentos de texto.

  • Ctrl + C: Copiar el texto o elemento seleccionado.
  • Ctrl + X: Cortar el texto o elemento seleccionado.
  • Ctrl + V: Pegar el texto o elemento copiado/cortado.
  • Ctrl + Z: Deshacer la última acción.
  • Ctrl + Y: Rehacer la última acción.
  • Ctrl + N: Crear un nuevo documento en blanco.
  • Ctrl + A: Seleccionar todo el contenido del documento.
  • Ctrl + G: Guardar el documento actual.
  • Ctrl + P: Abrir la ventana de impresión.
  • Ctrl + B: Poner el texto seleccionado en negrita.
  • Ctrl + K: Poner el texto seleccionado en cursiva.
  • Ctrl + S: Subrayar el texto seleccionado.
  • Ctrl + E: Centrar el párrafo seleccionado.
  • Ctrl + J: Justificar el párrafo seleccionado a derecha e izquierda.
  • Ctrl + L: Reemplazar (menú «Buscar y reemplazar»).
  • Ctrl + R: Alinear el párrafo seleccionado a la derecha.
  • Ctrl + F: Abrir el panel de búsqueda.
  • Ctrl + U: Abre un nuevo documento en blanco.
  • Ctrl + T: Centrar. Alinear el texto al centro.
  • Ctrl + D: Abrir el cuadro de diálogo Fuente.
  • Ctrl + H: Tabular texto.
  • Ctrl + M: Cambiar la tipografía (ventana «Fuente»).
  • Ctrl + Q: Alinear texto a la izquierda.
  • Ctrl + R: Cerrar el documento.
  • Ctrl + S: Subrayar texto.

Funciones avanzadas con atajos de teclado en Word

Entre la gran variedad de comandos que podamos utilizar en Word, existen algunos más complejos, los que realizan acciones menos comunes, pero igualmente útiles a la hora de desarrollar nuestros documentos de texto.

  • Ctrl + Shift + F: Cambiar la fuente.
  • Ctrl + Shift + W: Aplicar estilos guardados.
  • Ctrl + Shift + >: Aumentar el tamaño de la fuente.
  • Ctrl + <: Reduce el tamaño de la fuente.
  • Ctrl + >: Aumenta el tamaño de la fuente.
  • Ctrl + 1: Interlineado x1 (sencillo).
  • Ctrl + 2: Interlineado x2 (doble).
  • Ctrl + 5: Interlineado x1,5.
  • Ctrl + Supr: Eliminar una palabra a la derecha del cursor.
  • Ctrl + Retroceso: Eliminar una palabra a la izquierda del cursor.
  • Ctrl + Re Pág: Retrocede a la página anterior.
  • Ctrl + Av Pág: Avanzar a la página siguiente.
  • Ctrl + Flecha izquierda: Desplaza el cursor a la siguiente palabra a la izquierda.
  • Ctrl + Flecha derecha: Desplaza el cursor a la siguiente palabra a la derecha.
  • ALT + F4: Cierra Word.
  • ALT + F5: Restaura el tamaño de la ventana de Word.
  • ALT + F11: Abre una ventana con el código de Visual Basic.
  • Ctrl + Mayús + F12: Abre la ventana de Imprimir.
  • Ctrl + Alt + F1: Abre la ventana de Información de sistema de Windows.
  • Ctrl + Alt + F2: Abre la ventana del explorador para abrir documento.
  • Alt + C1: Abre la pestaña de Diseño, modificar márgenes, orientación, sangría y espaciado.
  • Ctrl + Fin: Ve al final del documento.
  • Ctrl + Inicio: Ve al principio del documento.
  • Ctrl + Alt + O: Inserta una nota al pié de página.
  • Ctrl + Alt + D: Inserta una nota al final.
  • Alt + Mayús + >: Ve a la siguiente nota al pié de página.
  • ALT + Mayús + <: Ve a la anterior nota al pié de página.

Atajos de Teclado en Microsoft Excel

Estos atajos son útiles para navegar y editar hojas de cálculo.

Gestión de archivos

  • Ctrl + N: Nuevo libro de trabajo
  • Ctrl + O: Abrir archivo
  • Ctrl + S: Guardar
  • F12: Guardar como
  • Ctrl + P: Imprimir

Edición

  • Ctrl + C: Copiar
  • Ctrl + X: Cortar
  • Ctrl + V: Pegar
  • Ctrl + Z: Deshacer
  • Ctrl + Y: Rehacer
  • Delete: Borrar contenido de celda
  • Ctrl + F: Buscar
  • Ctrl + H: Reemplazar

Navegación

  • F2: Editar celda seleccionada
  • Ctrl + Flechas: Mover al final del rango continuo (arriba, abajo, izquierda, derecha)
  • Inicio: Ir al principio de la fila
  • Ctrl + Inicio: Ir al principio del libro (A1)
  • Ctrl + Fin: Ir a la última celda utilizada

Selección

  • Ctrl + Barra espaciadora: Seleccionar columna completa
  • Shift + Barra espaciadora: Seleccionar fila completa
  • Ctrl + Shift + Flechas: Seleccionar rangos contiguos
  • Ctrl + A: Seleccionar todo

Fórmulas y celdas

  • = (igual): Comenzar una fórmula
  • Alt + =: Autosuma
  • Ctrl + ;: Insertar la fecha actual
  • Ctrl + Shift + :: Insertar la hora actual
  • F4 (al escribir fórmulas): Cambiar entre referencias absolutas y relativas ($A$1, A$1, $A1, A1)

Comandos generales de teclado:

  • Alt + F4: Cerrar la ventana activa o apagar la computadora si no hay ventanas abiertas.
  • Alt + Tab: Alternar entre diferentes ventanas abiertas.
  • Ctrl + A: Seleccionar todo el texto o los elementos en una ventana.
  • Ctrl + C: Copiar texto o elementos seleccionados.
  • Ctrl + F: Buscar texto en un documento o navegador.
  • Ctrl + P: Imprimir el documento actual.
  • Ctrl + S: Guardar el archivo actual.
  • Ctrl + Shift + Esc: Abrir el Administrador de tareas.
  • Ctrl + V: Pegar texto o elementos copiados.
  • Ctrl + X: Cortar texto o elementos seleccionados.
  • Ctrl + Z: Deshacer la última acción realizada.
  • Esc: Salir de una aplicación o ventana.
  • F1: Abrir la ayuda de la aplicación o sistema operativo.
  • Windows + E: Abrir el Explorador de archivos.
  • Windows + L: Bloquear la computadora.
  • Windows + R: Abrir la ventana de Ejecutar.
  • Windows + S: Abrir la barra de búsqueda de Windows.
